# extract-raw-url

Chrome 拡張機能の練習

Amazon の商品ページの長い URL を綺麗にします。

## How to setup

以下の手順に従ってローカルから拡張機能を追加します。

https://developer.chrome.com/docs/extensions/mv3/getstarted/development-basics/

## How to use

- Amazon の商品ページを開きます。
- 右クリックで `Amazonの綺麗なURLを開く` というメニューが表示されるのでクリックします。
- 別タブで、「綺麗なURL」として開かれます。
